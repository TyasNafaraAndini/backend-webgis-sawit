# 🌍 WebGIS Plantation Management System

## 📌 Description
This project is a web-based Geographic Information System (GIS) designed to manage and visualize plantation data.

## 🚀 Features
- Display plantation map using GeoJSON
- Manage spatial and non-spatial data
- RESTful API integration
- Data validation system

## 🛠 Tech Stack
- Laravel
- PostgreSQL
- GeoJSON
